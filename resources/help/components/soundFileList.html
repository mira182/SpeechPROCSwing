Sound file list

On the left part of application window is list of sound files for which you can start calculation of speech parameters. In this list you can select as many files as you want. Every line shows name of the sound file, absolute path to the file and its size.

Search field

On the top of this panel is Search text field so you can serch for specific sound files in the list. It searches the text you put in the field in names of the files in the list.

List buttons

Below the list you can see 2 buttons. First with plus icon is for adding sound files into list. The other one with the minus icon is for removing the files from the list.
