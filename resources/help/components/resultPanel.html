Buttons

In the bottom part of result panel are buttons for showing the result details.

Result details

When you push button Result details it will open separate window with the values of all selected result line. The result window is specific according to calculated speech parameters.

There are 3 specific windows showing the details:

  • Voice report details
  • Formants details
  • Intensity details

Every window with result details contains button for showing graph. Graph is also specific according to calculated speech parameters.
