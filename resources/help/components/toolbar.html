Toolbar

Toolbar contains few buttons to make the access to main functions of the application easier.

Add sound files

This buttons allows the user to add sound file to the sound file list

Save as

Settings

This button shows the settings window

Praat script

This allows the user to run Praat scripts.

Matlab script

This allows the user to run Matlab scripts.

Octave script

This allows the user to run Octave scripts.

Formants

This allows the user to calculate Formants.

Voice report

This allows the user to calculate Voice report.

Intensity

This allows the user to calculate Intensity.

Help

This shows Help window for the application.
