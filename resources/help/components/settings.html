Settings

Settings window allows the user to set path to Praat, Matlab and Octave programs so application can run scripts for these programs.

Default settings

When the program starts there is default settings set. The default paths to the programs are showed in table below.

Linux Windows
Praat /usr/bin/praat C:\Program Files\praat.exe
Matlab /usr/bin/matlab C:\Program Files\MATLAB\MATLAB Production Server\R2015a\bin\matlab.exe
Octave /usr/bin/octave C:\Program Files\octave.exe
