Here you will find out how to calculate intensity and some other parameters related to it.

Loudness menu item

First you have to select Prosody item in menu Parameters as shown on image below. Then you need to select Loudness menu item to calculate intensity.

Calculation of formant listing starts for every selected sound file in sound file list. All the results are then shown in result panel. Dispalying of results is more detailed described in result panel description.

Default parameters

Name Value
Minimum pitch (Hz) 75
Time step (s) 0.01
