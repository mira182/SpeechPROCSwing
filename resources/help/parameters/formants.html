Formants

Here you will find out how to calculate formants and some other parameters related to formants.

Formants menu item

First you have to select Formants listing item in menu Parameters as shown on image below.

After click on Formant listing it shows dialog with input parameters for fformants calculation. The dialog is shown on picture below.

When you click on OK button in this dialog, calculation of formant listing starts for every selected sound file in sound file list. All the results are then shown in result panel. Dispalying of results is more detailed described in result panel description.

Default parameters

Name Value
Time step (s) 0.005
Maximum number of formants 3
Maximum formant (Hz) 5000 Hz
Window length (s) 0.025
Pre-emphasis from (Hz) 50

User can ofcourse change these values in parameters dialog.
