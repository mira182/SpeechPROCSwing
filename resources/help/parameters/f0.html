Fundamental frequency

Here you will find out how to calculate fundamental frequency and some other parameters related to it.

Fundamental frequency menu item

First you have to select Voice report item in menu Parameters as shown on image below.

After click on Voice report it shows dialog with input parameters for fundamental frequency calculation. The dialog is shown on picture below.

When you click on OK button in this dialog calculation of voice report starts for every selected sound file in sound file list. All the results are then shown in result panel. Dispalying of results is more detailed described in result panel description.

Default parameters

Name Value
Time step (s) 0.05
Pitch min (Hz) 75
Pitch max (Hz) 600
